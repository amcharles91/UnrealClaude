## VibeUE

A substantial number of the native MCP tools in this plugin
(`Source/UnrealClaude/Private/MCP/Tools/MCPTool_*.cpp`) were **ported and
adapted** from the service implementations in **VibeUE**. The MCP tool registry
and several tool designs (animation, Niagara, audio/SoundCue/MetaSound,
landscape, foliage, StateTree, gameplay tags, enum/struct, UMG, and others)
